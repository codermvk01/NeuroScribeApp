\# NeuroScribe Plus



Early cognitive risk screening application.



\## Architecture

\- Frontend: Expo (React Native)

\- Backend: Node.js + Express

\- ML Server: FastAPI (Python)

\- Database: MongoDB



\## Folder Structure

frontend/   → Mobile app UI  

backend/    → APIs, Auth, DB, Reports  

ml-server/  → ML models \& scoring  

docs/       → Documentation
